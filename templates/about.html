About Diet Impact Analyser

A full-stack data science project built to analyse dietary habits using Python, Flask, and Chart.js

Project Overview

The Diet Impact Analyser is a full-stack web application that helps users understand the nutritional impact of their daily food intake. Users log their meals, and the system retrieves nutritional data, performs calculations using data science techniques, and visualises the results through interactive charts.

🎯 Objectives
  • Secure multi-user authentication with per-user diet databases
  • Log daily food consumption with date tagging and quantity entry
  • Calculate total caloric and nutrient intake using Pandas and NumPy
  • Compare results against Recommended Daily Intake (RDI) values
  • Visualise nutrition and historical trends using Chart.js (6+ chart types)
  • Generate personalised dietary suggestions based on profile BMI/TDEE
🏗️ Architecture

The system follows a clean 3-tier architecture:

  • Presentation Layer: HTML, CSS (Bootstrap 5), JavaScript
  • Application Layer: Python Flask backend with REST API
  • Data Layer: CSV dataset (90 food items) + Pandas/NumPy analysis
📊 Data Analysis Logic
  1. Nutrition data is loaded from a CSV dataset using Pandas
  2. User entries (food_id + quantity) are matched against the dataset
  3. Nutrients are scaled: value × (quantity / 100)
  4. NumPy aggregates totals across all food items
  5. Comparison calculates % of RDI for each nutrient
  6. Rule-based engine generates improvement suggestions

Fetch all logs for current user
POST /api/analyse
Analyse submitted food entries
POST /api/profile
Update height/weight/age/gender
GET /api/foods/search?q=
Autocomplete food search
GET /api/rdi
Return RDI reference values
